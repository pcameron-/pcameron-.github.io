Software Rasterizer

This is a small project to show that I can write a 3D software rasterizer without any other 3D API's.

This project includes:

  • Clipping
  • Culling
  • Bilinear Interpolation
  • Hyperbolic Interpolation (Floor texture)
  • Lighting
  • Shadows

Note: This project does not use hardware rasterization. It uses a set pixel function in software mode.
